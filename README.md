# MatruSnehHealth
Android app for maternal and child health.
# 🤰 MatruSneh Health (ಮಾತೃ ಸ್ನೇಹ ಹೆಲ್ತ್)

A bilingual Android application designed to support pregnant mothers with pregnancy tracking, baby growth monitoring, kick counting, nutrition guidance, and health awareness.

![Platform](https://img.shields.io/badge/Platform-Android-green)
![Language](https://img.shields.io/badge/Language-Kotlin-purple)
![Architecture](https://img.shields.io/badge/Architecture-MVVM-blue)

---

# ✨ Features

- 👶 Baby Growth Tracker
- 👣 Kick Counter
- 🩺 Pregnancy Check-up Guidance
- 🥗 Nutrition Support
- ⚠️ Danger Signs Awareness
- 🌐 English & Kannada Language Support

---

# 🛠 Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM
- **UI:** Material Design + ViewBinding
- **Database:** Room Database
- **Background Tasks:** WorkManager
- **Jetpack:** LiveData, ViewModel
- **SDK:** Android SDK 34
- **JDK:** 17

# 🚀 Getting Started

## Prerequisites

- Android Studio Iguana+
- JDK 17
- Android SDK 34

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/MatruSnehHealth.git
```

1. Open project in Android Studio  
2. Sync Gradle  
3. Run on Emulator or Physical Device
